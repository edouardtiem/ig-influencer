## Current State

Fanvue daily posting is **automated** via GitHub Actions. Posts go out at 17h with explicit content generated by ComfyUI.

### How It Works

```
GitHub Action (17h) ‚Üí ComfyUI generation ‚Üí Fanvue API upload ‚Üí Post live
```

### Active Configuration

| Component | Value |
|-----------|-------|
| **Schedule** | 17h daily |
| **Content** | Explicit (allowed on Fanvue) |
| **API** | Fanvue multipart upload |
| **Vault** | elena-feed |

## API Notes

### Correct Upload Format

```javascript
// POST /v1/posts (multipart/form-data)
{
  content: string,      // not "text"
  media_urls: array,    // not "mediaUrls"
  is_premium: boolean   // not "audience"
}
```
